# 值日提醒/duty_reminder
一个值日提醒软件/A duty reminder app

如你所见，一个值日提醒软件，本软件开发之初就是用在校园大屏上的提醒软件\
其实作者十分的懒，本来想要在广袤的互联网上寻找一个能够提醒值日的软件\
但没找到几个，为数不多的几个也不符合我的要求\
此时我看到了最近ai更新得挺厉害\
于是☝🤓\
这个软件就以AI为主，作者为辅而诞生了\
所以这个软件基本上就是用AI写的，纯AI低人工(bushi\
鉴于广大的互联网没有几个这样的软件，我觉得干脆发出来得了(\
用的是python写的，源代码全在一个文件里，发行版会有单文件和目录版\
单文件便于移动，但是慢(我也不知道为啥nuitka打包的一堆问题，所以用PyInstaller)\
目录版的快，但是不方便移动\
如果有大佬知道怎样提升打开速度的话谢谢大佬能帮我这个小白改一下🙏\
里面肯定会有一大堆的bug，有些修不了的已经注释掉了(\
~~比如语音提醒~~\
各位大佬们如果看到我这操作请补药喷我口牙\
谢谢了（

***

以下是使用教程:\
第一步，打开软件\
第二步，添加人员，可以使用excel添加，用excel时要注意第一行不识别，从第二行开始，并且只识别第一列\
第三步，调整提醒时间，这个提醒时间是提醒第二天人员的时间，在到达提醒时间后，会提醒明天xxx的人员值日\
总体使用步骤就这样\
